Industrial Project (234313)

Anomaly Detection using Denoising Diffusion Models.

Start scrolling!

Goals


  • To create a synthetic documents generator which will be used for extracting dataset for training deep learning model under the domain of documents understanding.
  • For each label we have in the original document we want to recognize its location in the synthesized image.

Methodology


Blender Scene Setup

  • Adding the meshes to the scene, two plane meshes one is the document and the other is the floor
  • Scene definition: in order to make the scene look realistic we define the materials of the objects, the lightning and the camera.
  • This part allows us to control the objects, lights and camera in the coding part and randomize their location and the intensity of the light.

The Paper Shape

  • Applying the document image as a texture to the object
  • Changing the shape of the paper, we support multiple options for the shape:
    • - Flat paper
    - Folded paper
    - Wrinkled paper
    - Bended paper

Extracting the labels algorithm

  • Divide the mesh faces to triangles (tessellation)

    • The mesh geometry before

    The mesh geometry after


  • Store the UV coordinates for each corner of the triangle in a KDtree

    • The triangle corners

    The UV coordinates of the corners

  • Map the bounding box points of a label ( for each point ):
    • -- Using the KDtree find the uv coordinate of the triangle corner which is the closest to the point's uv coordinate
    -- Find the triangle that contains the point.
    -- Find the point's barycentric coordinate relative to the traingle
    -- Find the local coordinate of the point using interpolation
    -- Find the world space coordinate of the point
    -- Find the camedra space coordinate of the point
  • Converting the bounding box points from camera space coordinates to the render space coordinates

    • Camera Space View

    Render Space


Automation And Parameters Randomization

  • Allows us to automate the render generation
  • Taking pictures of our object from different angles by moving the camera around the scene. There is three loops, each one of them modifies the angles of the camera, the first is the distance from the object, the second is the rotation along the X axis and the last is rotation from 0 to 360 degrees around the object.
  • We added a mode which supports moving the camera randomly
  • Randomize a texture for the environment and the floor
  • Randomization of the light intensity, color and location
  • The user can choose the shape of the paper and the intensity level of the wrinkles/ bending angle/ folding angle or a mix.
  • We also support adding an object which creates shadow on the document paper

Testing


  • We added a feature to draw the bounding box around each labeled word to help us check the results.
