Industrial Project (234313)

Anomaly Detection using Denoising Diffusion Models.

Start scrolling!

Goals


  • In this project we will attempt to develop a POC for detecting anomalies in images based on the ability or inability of a DDM (Denoising Diffusion Model) to reconstruct them.
  • The results of the POC will then determine whether there is grounds for further research for using Denoising Diffusion Models for Anomaly Detection problems.

Methodology


Anomaly Detection Pipeline

  • Add random gaussian noise to an image.
  • Denoise the noised image using a DDM and reconstruct the image.
  • Calculate the “difference” between the original image and the reconstructed image and use the result to determine an anomality score.

Note that the results were consistently better with object type categories rather than with textures. We anticipated this result, since the task of reconstructing images that feature items with an inherently random structure is typically harder for a denoising diffusion model, as it cannot precisely anticipate and reconstruct all of the random features that were degraded after the noise was added to the original image and it ends up making up extra features on its own typically.
